Tema 1 - Conceptos básicos en Javascript

¿Que es Javascript?

Javascript es el responsable de controlar toda la interactividad del usuario en la web.

Para ver si esta funcionando o no lo podemos ver a traves de la consola. Esta la encontraremos haciendo clic derecho en la web y vamos a la pestañita "consola".

El Javascript se puede introducir de tres formas distintas:

  • Escribiriendolo directamente dentro del body del HTML con una etiqueta "script"
  • Escribiendolo en el head del HTML
  • Enlaçando un archivo .js

Tipos de datos básicos:

  • string
  • number
  • boolean
  • undefined
  • null
  • bigit
  • symbol

Declaración y inicialicación

  • var => se usa cada vez menos en el desarrollo moderno de Javascript debido a sus peculiaridades y potenciales problemas con el alcance de función y hoisting.
  • let => es preferible cuando se necesita una variable cuyo valor va a cambiar, como contadores en bucles, o valores que se reasignan en un bloque de código.
  • const => es la mejor opción para declarar variables que no deben cambiar después de su asignación inicial. Esto mejora la legibilidad del código y reduce la posibilidad de errorres inesperados.

Conceptos utilizados...

  • alert() => Para hacer que salte un popup con un mensaje de alerta
  • El console.log() => Para imprimir mensajes por consola
  • El console.warn() => Para imprimir mensajes de "warning" por consola
  • El console.info() => Para imprimir mensajes de información por consola
  • El console.error() => Para imprimir mensajes de error por consola
